# Jeunes Leaders de Demain

MVP prototype — programme éducatif complémentaire destiné aux jeunes de 12 à 17 ans au Québec.

Ce site est le prototype virtuel (moyenne fidélité) développé dans le cadre du cours ACT101 – Préparation à la création d'entreprise, Université de Sherbrooke, Hiver 2026.

## Ce que contient ce repo
- `index.html` — page web complète (landing page MVP + formulaire liste d'attente)
- `BMC_Jeunes_Leaders_Demain.html` — Canevas du modèle d'affaires (Business Model Canvas)

## Déploiement
Site hébergé via GitHub Pages : https://[ton-username].github.io/jeunes-leaders-demain

## Projet
Jeunes Leaders de Demain est un programme structuré en cohortes de 4 à 6 semaines visant à développer la communication, la confiance en soi, la gestion des émotions et l'ouverture culturelle chez les jeunes — là où l'école n'arrive pas toujours seule.
